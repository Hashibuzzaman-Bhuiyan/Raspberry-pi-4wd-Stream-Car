# Raspberry-pi-4wd-Stream-Car
4wd-Stream-Car made with Raspberry-pi 4b and pi camera V2

you have to create a folder"rb" in the main directory. Inside of the "rb" folder, you have to create "templates" folder and put the "html" file in in. Close all the folders. Now go to the "rb" folder and put your python file in the same directory where  you kept the "templates" folder.

here is a structure:


main ============> rb ====> templates ====> index.html
                      |
                      |===> robo.py
